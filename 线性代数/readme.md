# 线性代数

当前学校主要开设的线性代数课程包括：

* **线性代数（非书院版）**：使用梁鑫等编著的《线性代数入门》作为教材，自2019年开始启用。
* **线性代数（书院版）**：使用俞正光等编著的《线性代数与几何》作为教材，自2019年停止，2022年再次为书院启用。
* **线性代数（英）**：本人不了解。
* **线性代数（理科类）**：物理系学生学习的线性代数。

前两种分类非官方分类，仅为个人根据课程的限选要求与内容做的区分。本库暂未收集**高等线性代数选讲**的资源。
